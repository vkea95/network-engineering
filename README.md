# Network Engineering Learning System

Purpose:

1. Build strong engineering troubleshooting ability
2. Understand how the Internet actually works
3. Support backend / AI engineering with solid networking knowledge
4. Develop practical Linux + shell skills
5. Build foundation for web security awareness

Environment:

Mac (client)
↓
Linux Mint (server)

Learning Strategy:

Observe → Verify → Automate → Understand → Explore

Command line first  
Automation second  
Protocol third  
Security fourth

Key Skills:

Linux shell  
Network troubleshooting  
HTTP understanding  
Packet observation  
Automation tools
